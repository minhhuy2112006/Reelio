# Reelio
Reelio is an powered memory management application that transforms ordinary photo storage into meaningful memory experiences.

The app allows users to organize photos into themed collections, capture moments through a built-in Photo Booth, generate AI-powered recap stories, and explore memories using immersive 3D depth visualization.

Features
📂 Organize photos by custom themes and albums
🖼️ Apply creative frames and visual layouts
📸 Capture photo strips with Photo Booth mode
🎞️ Generate monthly, yearly, or event-based recaps
🌌 View photos in interactive 3D Depth View
🤖 AI-assisted memory curation
📤 Share memory collections and highlights
Vision

Memora aims to become a digital memory vault where users can preserve, relive, and share the stories behind their photos instead of simply storing them.

Every photo tells a story. Memora helps you relive it.
